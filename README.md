# An-lise-de-Viabilidade-Econ-mico-Financeira-de-Projetos
Projeto realizado com o uso do NotebookLM, utilizando prompts estratégicos para organizar conceitos financeiros introdutórios (VPL/NPV, TIR/IRR, Payback), gerar exemplos práticos e estruturar o conteúdo em resumos, tabelas e roteiros de estudo aplicáveis a projetos industriais.
